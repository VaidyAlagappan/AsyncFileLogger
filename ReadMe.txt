1. This project is developed using VS 2022 with .Net 8.0 framework
2. Created a console application
3. You can right click at the project file and create a 'docker support' file
4. You should have Docker Desktop installed in your local machine
5. Create a user account in "https://hub.docker.com/"
5. Create a repository in "https://hub.docker.com/"
6. Debug your application locally to check if it works.
7. Publish it to docker.  You need your username and password to the registry.
8. To test, user can pull the docker container and run it using docker desktop
9. Use command prompt
